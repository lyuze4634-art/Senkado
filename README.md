# 選考舵

## 公開サイト

本サイトは以下のURLで公開しています。
https://senkado.cc/

**選考舵** は、日本の就職活動で発生する企業ごとの選考状況や締切をまとめて管理するための Web アプリケーションです。

複数の企業に応募していると、MyPage、メール、説明会予約、ES提出、面接日程などの情報が分散しやすくなります。
このサイトでは、それらを一つの画面で整理し、「次に何をすればいいか」を確認しやすくすることを目的としています。

---

## 画面イメージ

トップページでは、登録した企業と最近の未完了タスクを一覧で表示します。
締切が近いものや過ぎているものを分かりやすく表示し、就職活動中の確認漏れを減らします。

デザインは、黒と白を基調にしたシンプルで落ち着いた雰囲気を意識しています。

## 制作目的

このプロジェクトは、就職活動の情報を自分で整理しやすくするために制作しました。
単なるメモではなく、企業ごとの進行状況や締切を一目で確認できる管理ツールを目指しています。
