# 選考舵（Senkado）

日本の就職活動における選考状況・次回アクション・締切日・企業リンクを一元管理するための Web アプリケーションです。

本システムは、複数企業の MyPage、メール、説明会システム、選考案内などに分散しやすい情報をまとめ、ユーザーが「次に何をすべきか」をすぐ確認できるようにすることを目的としています。

---

## 概要

**選考舵** は、就職活動中のユーザーが企業ごとの選考状況を管理するための PHP + MySQL 製の管理サイトです。

ログイン後、ユーザーは自分が登録した企業一覧を確認できます。各企業には、企業名、企業リンク、選考状態、次回アクション、締切日時、備考などを登録できます。

トップページでは、企業ごとの「最近の未完了アクション」を表示し、締切が近い項目や期限切れの項目を分かりやすく確認できます。

## 活動状態

活動状態は以下を想定しています。

* 未着手
* 進行中
* 完了
* キャンセル

期限切れかどうかは状態として保存せず、`due_at` と現在時刻を比較して自動判定します。

## トップページの表示ロジック

トップページでは、各企業ごとに最近の未完了アクションを1件表示します。

条件：

1. 現在ログイン中のユーザーの企業のみ表示する
2. 完了・キャンセル済みの活動は除外する
3. 締切日時が近い順に並べる
4. 各企業につき最も近い未完了活動を1件表示する
5. 未完了活動がない場合は「次回アクションなし」と表示する

表示例：

| 企業 | 表示内容                    |
| -- | ----------------------- |
| A社 | ES提出：2026-06-20 23:59   |
| B社 | 一次面接予約：2026-06-25 12:00 |
| C社 | 次回アクションなし               |

## セットアップ方法

### 1. リポジトリを配置する

XAMPP の `htdocs` フォルダ内にプロジェクトを配置します。

```text
htdocs/123/
```

### 2. MySQL データベースを作成する

phpMyAdmin などを使用して、以下のデータベースを作成します。

```text
job_hunt_manager
```

### 3. テーブルを作成する

`users`、`companies`、`activities` テーブルを作成します。

### 4. データベース接続情報を設定する

`PHP/config/db.php` に MySQL の接続情報を設定します。

```php
$host = 'localhost';
$dbname = 'job_hunt_manager';
$user = 'root';
$password = '';
```

### 5. XAMPP を起動する

Apache と MySQL を起動します。

### 6. ブラウザでアクセスする

```text
http://localhost/123/
```

## セキュリティ方針

本システムでは、就職活動に関する個人情報や企業情報を扱うため、以下の対策を行います。

* パスワードは平文で保存しない
* `password_hash()` でパスワードをハッシュ化する
* `password_verify()` でログイン認証を行う
* SQL は PDO のプリペアドステートメントを使用する
* ログイン必須ページではセッションを確認する
* 他ユーザーのデータにアクセスできないように `user_id` を確認する
* 画面出力時は `htmlspecialchars()` を使用する
* POST フォームには CSRF 対策を行う
* ログイン成功時に `session_regenerate_id(true)` を実行する
